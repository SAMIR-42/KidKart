KidKart 🛒
Fun Shopping App for Kids
A playful, kid-friendly fake shopping experience — like Flipkart but built just for children. Kids can browse toys, add to cart, and go through a fun checkout flow with cool animations, all in a safe, real-feeling environment.

💡 The Idea
Adults have Flipkart and Amazon. Kids always want to "shop" too. KidKart gives children a fun, animated shopping experience — completely safe, completely fake, completely awesome.

✨ Features

Browse kids products — toys, games, fun items
Add to cart with satisfying animations
Checkout flow that feels real
Fun, colorful UI designed for children
Smooth page transitions and interactions
Category-wise browsing — boys, pleasure sections


🎨 Design Philosophy
Everything is built to feel exciting for a child:

Big colorful buttons
Fun animations on every action
Simple navigation — no confusion
Real e-commerce feel without real transactions
